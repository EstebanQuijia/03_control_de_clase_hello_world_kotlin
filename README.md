# 003 Control de Clase Hello World Kotlin
## Cómo Ejecutarlo

1.  **Ejecutar la aplicación:** Localiza el archivo `Example.kt` y haz clic en el botón verde de "Play" (Run) que aparece en la barra superior de IntelliJ, o junto a la función `main` en el editor.
2.  **Acceder a los Endpoints:** Una vez que la aplicación esté ejecutándose (normalmente en `localhost:8080`), puedes acceder a los siguientes URLs en tu navegador web o con una herramienta como Postman:
    *   `http://localhost:8080/hello`
    *   `http://localhost:8080/square`
    *   `http://localhost:8080/triangle`
